# viksit-bharat-makkapakkagang-vocal-for-local
A creative project under Viksit Bharat – Vocal for Local, celebrating India’s endangered handicrafts — Lippan Kaam and Tholu Bommalata. This website spreads awareness about their origin, beauty, and heritage while promoting local artisans through digital innovation, AI interaction, and youth-driven cultural revival.Made by Makka Pakka Gang
